### Как запустить сервер на версии 56 на андроид?
1.Скачайте апк, архив и пидройд 3 (Пидройд 3 в Гугле).
2.Зайдите в пидройд 3 и найдите архив
3. Нажмите на ,, Запуск py" и на жёлтую кнопку 
4. Готово! Играйте в ABS V56!
